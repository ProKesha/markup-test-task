# Monblanproject markup test

One-page responsive markup based on the provided Figma screenshots.

## Features

- Responsive profile/posts layout.
- Grid and list view switcher.
- Third-party JavaScript datepicker via Flatpickr.
- Static HTML/CSS/JS, no build step required.

## Local preview

```bash
python3 -m http.server 4200
```

Open `http://127.0.0.1:4200/`.
